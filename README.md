# Covid-App

**Covid-App** is your go-to mobile application for up-to-date information on the Covid-19 pandemic. With real-time data pulled from reliable APIs, you can quickly and easily stay informed on the latest statistics for any country of your choosing, including confirmed cases, deaths, and recoveries.

In addition, **Covid-App** provides news articles from reputable sources to keep you informed on the latest developments in the Covid-19 world. You can also use the app to locate nearby hospitals, ensuring you have access to the medical care you need.

With a user-friendly interface and a commitment to accuracy and reliability, **Covid-App** is the perfect tool for anyone looking to stay informed during these challenging times.

---

## Features

- Real-time Covid-19 statistics by country
- News updates from reputable sources
- Nearby hospitals locator
- Clean and intuitive user interface
